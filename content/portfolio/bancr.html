bancr is the primary R client for the BANC connectome. It wraps the CAVE, SeaTable, Google Cloud Storage and synapse-query endpoints behind a small set of R functions that return tidy tables, neuron objects and ggplot scenes.

What you can use it for. Use it to pull the per-neuron metadata table, query connectivity and pre-computed influence, find cross-dataset NBLAST matches, read neuron meshes and skeletons, transform neurons into shared template spaces, and render canonical views. It is the natverse entry point to the BANC.

influencer is an R implementation of the linear-dynamical-systems influence model used throughout the BANC paper, wrapping the Drugowitsch lab’s ConnectomeInfluenceCalculator with native R and Python backends.

What you can use it for. Use it to estimate how strongly one neuron (or group of neurons) influences another across an entire connectome, a scalable measure of network distance. It caches the eigen-decomposition between seeds, so repeated queries against the same connectome are fast.

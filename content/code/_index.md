I develop open-source software for neuroanatomy and connectomics, mostly in R as part of the [natverse](https://natverse.org) and in Python. The main products are listed below; see my [GitHub](https://github.com/alexanderbates) for the rest.

### Brain and nerve cord (BANC) connectome

- **bancr** — the primary R client for the BANC connectome. [GitHub](https://github.com/natverse/bancr) · [Zenodo](https://doi.org/10.5281/zenodo.20350647)
- **influencer** — R implementation of the linear-dynamics influence model. [GitHub](https://github.com/natverse/influencer) · [Zenodo](https://doi.org/10.5281/zenodo.20350563)
- **ConnectomeInfluenceCalculator** — Python influence engine (Drugowitsch lab). [GitHub](https://github.com/DrugowitschLab/ConnectomeInfluenceCalculator) · [Zenodo](https://doi.org/10.5281/zenodo.15999929)
- **nat.ggplot** — publication-quality 2D rendering of neurons and meshes. [GitHub](https://github.com/natverse/nat.ggplot) · [Zenodo](https://doi.org/10.5281/zenodo.20350565)
- **bancpipeline** — the R + SLURM pipeline behind the BANC compiled data. [GitHub](https://github.com/htem/bancpipeline) · [Zenodo](https://doi.org/10.5281/zenodo.20350571)
- **BANC-project** — analysis code and figures for the BANC paper. [GitHub](https://github.com/htem/BANC-project) · [Zenodo](https://doi.org/10.5281/zenodo.20350641)

### natverse R packages

- **natverse** — the meta-package tying the ecosystem together. [GitHub](https://github.com/natverse/natverse)
- **nat** — the core NeuroAnatomy Toolbox for reading, analysing and plotting neurons. [GitHub](https://github.com/natverse/nat)
- **neuprintr** — query neuPrint connectome servers (e.g. the hemibrain). [GitHub](https://github.com/natverse/neuprintr)
- **neuronbridger** — work with NeuronBridge light-to-EM neuron matches. [GitHub](https://github.com/natverse/neuronbridger)
- **neuromorphr** — client for [NeuroMorpho.org](https://neuromorpho.org). [GitHub](https://github.com/natverse/neuromorphr)
- **insectbrainr** — client for the [Insect Brain Database](https://insectbraindb.org). [GitHub](https://github.com/natverse/insectbrainr)
- **mouselightr** — work with MouseLight whole-brain mouse neurons. [GitHub](https://github.com/natverse/mouselightr)

*Each package's documentation and tutorials are linked from its GitHub page; several are also written up in my [portfolio](/portfolio/).*
